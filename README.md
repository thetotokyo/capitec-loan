This is a React NextJS 16 project - Capitec Loan APP

## Getting Started

First, run the following command to make sure all depedencies are installed:

```bash
npm install
```


## Run Project

To run the development server:

```bash
npm run dev
```

To run the production server:

Build Project
```bash
npm run build
```

Build Docker Image
```bash
docker build -t capitec-loan .
```

Run container
```bash
docker run -p 3000:3000 capitec-loan
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
